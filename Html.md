# __ Structure of every documents__ :


* before everything in the first line:

```html
<!DOCTYPE html>
```

* html tags (all the html code must be in it):
* head tags
* body tags
* a title : shown in the top of the browser

```html
<html>

  <head>
    <title> Title </title>
  </head>

  <body>
  </body>

</html>
```
___
# __What goes in the ```<body> </body>``` __ :

### __Headings :__

```html
<h1> Heading1 </h1>
<h2> Heading2 </h2>
<h3> Heading3 </h3>
<h4> Heading4 </h4>
<h5> Heading5 </h5>
<h6> Heading6 </h6>
```
### __Paragraphs :__

```html
<p> Paragraph </p>
```
### __Unordered lists :__

```html
<ul>
  <li> item1 </li>
  <li> item2 </li>
  <li> item3 </li>
</ul>
```

### __Ordered lists :__

```html
<ol>
  <li> item1 </li>
  <li> item2 </li>
  <li> item3 </li>
</ol>
```

## __Links :__

Atributes:
- href : url of the Links
- target : where the link opens
  * "_blank" : opens in a new window


```html
<a href = "url" target = "value" >
printedOnScreen
</a>
```
